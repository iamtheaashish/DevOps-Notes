# This is a note in text document about how to use markdown(.md) file.
# Syntax for Markdown is different on different platforms.
# This is specific to GitHub Flavored Markdown (GFM)


==========


```md

```

```md

```

```md

```

```md

```

```md

```

# Table with 3 Columns and 3 Rows, Atleast 3 dashes to separate. 
# No matter if you have more than 3, it is going to render exactly the same.
```md

| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Row 1    | Data 1   | Data 2   |
| Row 2    | Data 3   | Data 4   |
| Row 3    | Data 5   | Data 6   |

```

# Text Alignment inside of Table, use colons.
# Header 3 will be right-aligned.
# Header 2 will be left-aligned.  
# Header 1 will be center-aligned.
```md

| Header 1 | Header 2 | Header 3 |
|:----------:|:----------|----------:|
| Row 1    | Data 1   | Data 2   |
| Row 2    | Data 3   | Data 4   |
| Row 3    | Data 5   | Data 6   |

```
